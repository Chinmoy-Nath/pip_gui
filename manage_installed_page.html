Manage Installed Packages Page¶

It provided options for dealing with already installed packages. Most common ways in which user wants to deal with already installed packages is ei:ther to uninstall or update it. Therefore, currently two options are implemented:

  1. Update Installed Packages Page
  2. Uninstall Package Page

The page layout is similar to Install Page layout.

Update Installed Packages Page¶

Similar to InstallPage.InstallFromPyPI page.

Uninstall Package Page¶

The page layout is very similar to Update Installed Packages Page except the treeview has only two headers : package name and installed version.
